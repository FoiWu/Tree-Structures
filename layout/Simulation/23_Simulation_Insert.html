產生2-3 Tree的圖的步驟:
(I)先取得一筆資料
(II)按下"Show 2-3 Tree"按鈕即可一步一步的產生一棵2-3 tree
(III)若遇到overflow的情況，請按下"Solve Overflow"按鈕

可以使用兩種方式得到一筆資料以產生一個2-3 Tree的圖:

➢ 按下"Random Data"的按鈕,隨機產生一組data:

➢ 按下"Input data"的按鈕後,輸入8個數字,最後按下"Submit":

若有需要記錄當前步驟以提供下一個步驟做參考，請按下方的"Get this step"按鈕，
並選取右邊"previous step"字樣（藍色方塊），即可觀賞記錄的步驟畫面。
強烈建議動畫結束後再按"Get this step"按鈕，以確保記錄之步驟是完整的。
