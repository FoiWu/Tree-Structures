產生Binary Search Tree的步驟:

  • 先取得一筆資料
  • 再按下"Show Tree"按鈕即可產生一個Binary Search Tree
  • 最後按下"Traversal"按鈕後,可以得到pre-order,in-order,post-order和breadth first

可以使用兩種方式得到一筆資料以產生一個Binary Search Tree:

➢ 按下"Random Data"的按鈕,隨機產生一組data

➢ 按下"Input data"的按鈕,輸入完數字後按下"Submit"

由上往下依序比較節點的值，將小的值放左子樹，將大的值放右子樹
