Red-Black Tree的特性:
(I)Root node一定是黑色的
(II)從Root到Leaf的每個路徑,皆有相同數目的黑色node
(III)不可以有連續兩個Red node 

    每新增一個新的node X (每一個新增的Node一律為紅色), 即檢查其parent node的顏色?
    
    (一)Parent node為黑色 --> End
    
    (二)Parent node為紅色 --> 檢查其uncle node的顏色?
    
                        (1)Uncle node為紅色
                            -->  Parent node和Uncle node皆改為黑色, ancestor node再改為紅色
                       
                          
                        (2)Uncle node為黑色或沒有uncle node
                            --> 若新node為parent node的左(右)節點, 
                                   檢查parent node是否也為ancestor node的左(右)節點?
                        
                                    (i)是(Ancestor node到其新node的路徑上沒有轉彎)
                                            -->Single rotate完, 原Parent node改為黑色, 原ancestor node改為紅色
                             
                            
                                    (ii)否(Ancestor node到其新node的路徑上有轉彎)
                                            -->Double rotate完, 新node改為黑色, 原ancestor node改為紅色
